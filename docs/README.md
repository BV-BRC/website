##### Homepage/Bootstrap Build Process

1. Make edits to `patric_bootstrap.scss` and `patric_bootstrap_custom.scss`
2. Run SASS on main file: `npm run build:bootstrap`
3. Commit both `.scss` and compiled `.css`

##### Homepage Javascript
